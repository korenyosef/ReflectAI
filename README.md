# ReflectAI ✨

**Helping kids think before they type! 💡**

## What is ReflectAI? 🚀
ReflectAI is a Python project made for the **Nitzanim 2026 Hackathon!** 🎉 We want to help young kids learn how to communicate better online and stop cyberbullying before it even happens. 🛡️

Parents install this software on their kid's device. It listens to the keyboard in the background. If the kid types something hurtful, offensive, or mean, the app pauses their typing and shows a friendly pop-up. 💬 It asks them to think about what they wrote and uses AI to suggest a nicer, more positive way to say the same thing. 🌈

## How it Works ⚙️
1. 👂 **Listen:** The app quietly checks the text the user is typing on the keyboard.
2. 🧠 **Analyze:** The text goes to a secure AI model (`Qwen/Qwen2.5-7B-Instruct`).
3. 🛑 **Reflect:** If the AI detects offensive words or an angry tone, a friendly pop-up appears on the screen.
4. 💖 **Improve:** The app asks a reflection question (like *"Are you sure you want to say it like this?"*) and offers a kinder alternative generated by the AI.

## Built With 🛠️
* 🐍 **Python:** The core project is written entirely in Python.
* 🤖 **AI Model:** We use a smart and secure AI model (`Qwen/Qwen2.5-7B-Instruct`) to analyze the text and generate better responses safely.

## Why ReflectAI? 🌱
Kids don't always mean to be hurtful. Sometimes they are just angry or type too fast. 🏃‍♂️💨 ReflectAI gives them a small pause to calm down, reflect on their words, and choose a better path. It teaches them self-improvement and empathy instead of just blocking them! 🌟
